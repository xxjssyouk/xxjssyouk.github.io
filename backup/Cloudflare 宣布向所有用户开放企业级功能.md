Cloudflare 宣布将在未来一年内向所有客户开放几乎所有功能，无需企业账户即可购买使用。该公司表示将取消"企业专属"功能概念，让任何规模的团队都能使用最先进的配置选项。

首个开放功能是仪表板单点登录（SSO），现已向所有计划用户免费提供。Cloudflare 还新增 GitHub 社交登录支持，并承诺未来所有新功能都将遵循这一开放模式，无需销售团队介入即可自助购买。
